# 📊 Telecom Customer Churn Prediction

## 📌 Overview

This project predicts whether a telecom customer is likely to churn based on customer usage patterns and account information. The goal is to help telecom companies identify at-risk customers and improve retention strategies.

---

## 🎯 Business Objective

Customer churn is a major challenge in the telecom industry, leading to significant revenue loss. This project builds a machine learning model to predict churn and support data-driven decision-making.

---

## 🧠 Project Workflow

* Data Cleaning & Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Engineering
* Model Building (Multiple ML Models)
* Model Evaluation (Accuracy, Precision, Recall, F1-score, ROC-AUC)
* Deployment using Streamlit

---

## ⚙️ Technologies Used

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
* Streamlit

---

## 🤖 Machine Learning Models Used

* Logistic Regression
* Decision Tree
* Random Forest
* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)
* Naive Bayes

---

## 📊 Key Insights

* Customers with high customer service calls are more likely to churn
* International plan users show higher churn probability
* Usage patterns significantly influence churn

---

## 🚀 Deployment

The model is deployed using Streamlit, allowing real-time predictions by entering customer details.

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

---

## 📈 Output

* Predicts whether a customer will churn or not
* Displays churn probability
